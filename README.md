The Aviation Reservation is a simple and interactive **C++ console application** that allows users to view available flights and reserve seats. It uses file handling to store flight data and updates seat availability in real time. This project is ideal for learning OOP concepts, file operations, and basic data management techniques.

- Displays all available flights with destination, departure, and seat count

- Uses system commands for smooth console interaction
